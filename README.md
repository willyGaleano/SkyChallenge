## Config
Crear los archivos .env, .stag.env y .prod.env con las variables:

```bash

```

Tambien crear el archivo rabbitmq.env con las variables: 

```bash
RABBITMQ_USER=
RABBITMQ_PASSWORD=
RABBITMQ_HOST=
RABBITMQ_QUEUE_NAME=
```
